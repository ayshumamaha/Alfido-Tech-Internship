# Data Structures Implementation in C

## Overview

This project demonstrates the implementation of fundamental data structures using the C programming language. The application is menu-driven and allows users to perform various operations on Linked Lists, Stacks, Queues, and Binary Search Trees.

The project is designed to strengthen the understanding of data structure concepts and their practical implementation.

---

## Features

- Linked List
  - Insert Node
  - Delete Node
  - Search Node
  - Display List

- Stack
  - Push
  - Pop
  - Peek
  - Display Stack

- Queue
  - Enqueue
  - Dequeue
  - Front Element
  - Display Queue

- Binary Search Tree
  - Insert Node
  - Search Node
  - Inorder Traversal
  - Preorder Traversal
  - Postorder Traversal
