# Log File Management System in C

## Overview

The Log File Management System is a menu-driven C application developed to demonstrate file handling and system utility concepts. It enables users to create, manage, search, back up, and maintain log files efficiently.

The application stores log records permanently using text files.

---

## Features

- Create Log File
- Add Log Entry
- View Log Entries
- Search Log Entry
- Count Log Entries
- File Statistics
- Backup Log File
- Clear Log File
- Delete Log File

---

## Technologies Used

- C Programming
- File Handling
- GCC Compiler
